## Overview

Marketing was bottlenecked on design hand-offs. Every new landing took 1.5 months of figma → review → code → review → ship. Replaced it with a Markdown-first pipeline owners could drive themselves; designers own polish, not assembly.

## What I owned

- Block library design — every block paired with an MD frontmatter shape that non-designers could write without seeing Figma
- MCP chat embedded in the live page — owners ask "make this CTA stronger" and edits land in the DOM, the chat commits to git
- Brand audit: kept what worked, retired the rest, rewrote spacing tokens so default MD assembly looked right out of the box
- Designer-as-polish workflow: any owner-edited landing comes through me for typography, hierarchy, motion — but the assembly is theirs

## Outcome

Median cycle: 1.5 months → 4 days. Designers freed up for cases that need design, not landings that need execution.
