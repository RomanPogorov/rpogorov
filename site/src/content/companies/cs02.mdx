## TL;DR

- Stood up a **shadCN-based design system** end-to-end — Figma + code + Storybook in lockstep, plus a Claude skill that pulled components straight out of Figma before that tooling existed elsewhere.
- Built a markdown-first **landing platform** with an in-browser DOM-aware Claude chat — time-to-landing dropped from 1.5 months to 2–3 days. The whole company website now runs on it.
- Generators on top of the same primitives — **presentation generator** for any teammate, **kudos image generator** trained on the brand mascot, embedded analytics on every page.
- Vibe-coded the **FireCamp conference app** in a week — voting, schedule, subscriptions. Became the proof point that convinced C-level the code-first approach was net positive, not a side experiment.

- **Design system: Figma to Storybook** — shadCN-based, FHIR-aware variants (resource pickers, value-set autocompletes), bidirectional sync via Code Connect.

- **Presentation generator** — internal tool: any teammate types markdown, gets a brand-consistent deck.
- **Embedded analytics on every page** — engagement signals captured in-flow without retro instrumentation.
