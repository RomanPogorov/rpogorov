## What it does

- Reads design tokens and component specs from Figma via the API + Code Connect
- Lets Claude propose changes in natural language, executes edits as Figma plugin commands
- Roundtrips back: edits in code reflect in Figma so designers see the change in their normal tool
- Eliminates "designer changed the spec, dev didn't see it" drift — both sides edit one source

## Why

Designer-to-engineer handoff was the slowest part of any token or component change. Every spec tweak became a ticket, a meeting, a re-export. Vibe-coding the pipeline collapses that into one conversation.
